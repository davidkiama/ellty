# Page Selection React App

A simple React app for selecting pages using custom checkboxes.  
Built with Create React App and styled using SCSS.

## Features

- "All Pages" option toggles all selectable pages (2–4)
- Individual toggling for Pages 2–4
- Page 1 is always selected and disabled
- Shows alert on submit with selected pages
- Uses custom SVG icons for checkbox states

## Scripts

```bash
npm install
npm start
```
